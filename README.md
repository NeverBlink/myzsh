## myzsh
Personalized zsh setup with suggestions, auto-complete and powerlevel10k

# Installation
1. git clone https://github.com/NeverBlink/myzsh.git
2. cd myzsh
3. chmod +x setup_zsh.sh
4. ./setup_zsh.sh
